The Beacon interface is used to schedule an asynchronous and non-blocking request to a web server. Beacon接口： 用于给服务器端 发送异步，非阻赛的请求 Beacon requests use the HTTP POST method and requests typically do not require a response. Beacon请求采用 POST请求，且通常该请求不要求要有response Requests are guaranteed to be initiated before a page is unloaded and they are run to completion, 页面未加载，和它们已经 without requiring a blocking request (for example XMLHttpRequest).
