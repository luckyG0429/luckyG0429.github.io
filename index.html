致给自己：

从2015年初开始接触前端，从一个迷迷糊糊的家伙变成了一个有明确方向的家伙

三年以来的很多日子，工作中积累，日常中学习，去拓展自己的更多技能。响应式， js小动画， css动画，3D效果，数据可视化，3D动画... 刚入行时候的知识网图很简单- "js + css + html + jquery"

后来...

  • + http
  • + 浏览器 + 性能
  • + 框架 + nodejs
  • + 用户体验 + 交互设计

一路就这样走过来了，马上就要满3年了，本以为已经是老手的自己，一定对很多东西已经是得心应手，熟练于心。就在自己信心满满，开始满腔热血的去投入工作中， 才发现：很多的效果，代码，逻辑，自己都曾经花心思做过的东西却没有做过任何的笔记和备份，还觉得自己那时候很厉害。自己的这种满足和自大，让自己现在又得花时间去重新研究，毕竟很多东西，灵感是一时的，只有总结才会让自己更深刻， 所以我现在起，要边用边记。

就当是对自己的一种成长记录，和鼓励。

我想自己真的决定要好好沉淀一下自己的所学所得
