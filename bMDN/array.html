Arrary - 常用的内置方法(1)

关于Array,我们常常会使用数组的内置方法，很多时候我们并不知道我们使用的方法是否合适，从以下使用方法的三个维度来判断应该使用哪个方法能够更好的实现，他们分别是 入参， 返回值， 是否改变原数组。

根据条件过滤数组值 ⤧ filter() & find()

  • [不改变原数组] —— filter() 入参：Fn —— 你可以自己定义一个函数fn，函数必须要有return true或者false 返回：filter遍历整个数组，然后将结果为true的值，组成一个新的数组对象。 不改变原数组 
    •                 复制
               var arr = [1,4,5,10];
                console.log('------------------')
                console.log('filter实现：返回大于5的数组');
                var result = arr.filter(function(i){
                         return i>5
                });
                 console.log(result);
  • [不改变原数组] —— find() 入参：Fn —— 你可以自己定义一个函数fn，函数必须要有return true或者false 返回：返回的是满足提供函数要求的第一个元素。否则返回undefined 不改变原数组 
    •                 复制
               var arr = [1,4,5,10];
                console.log('------------------')
                console.log('find实现：返回大于5的数组');
                var result = arr.filter(function(i){
                         return i>=5
                });
                 console.log(result);

数组截取 ⤧ slice() & splice()

  • [不改变原数组] —— slice() - 浅拷贝 入参: 数组下标的开始位置和结束位置 返回: 一个新的数组对象 不改变原数组 
    •                 复制
               var arr = ['ni','zhen','hao',10, 'kan'];
                console.log('------------------');
                console.log('slice实现：截取数组，从第2个开始，截取3个');
                var result = arr.slice(1, 4)
                 console.log(result);
  • [改变原数组] —— splice() 入参：数组下标，删除元素个数，你想添加的值 返回： 数组对象。 如果是删除，返回的被删除的值的数组。如果是添加或者在原地进行修改，返回的是空数组。 改变原数组 
    •                 复制
               var arr = ['ni','zhen','hao',10, 'kan'];
                console.log('------------------');
                console.log('splice实现：截取数组，从第2个开始，截取3个');
                var result = arr.splice(1, 3)
                 console.log(result);

合并数组 ⤧ concat()

  • [不改变原数组] —— concat() 入参： 数组列表 返回： 一个新的数组 不改变原数组 
    •                 复制
               var arr = ['ni','zhen','hao',10, 'kan'],
                   arr1 = [10,11,13],
                   arr2 = ["你","好呀", "!"];
                console.log('------------------');
                console.log('contact实现：多个数组合并');
                var result = arr.concat(arr1, arr2)
                 console.log(result);

遍历数组 ⤧ forEach() & map()

  • [不改变原数组] —— forEach() 入参：Fn 返回： 空 
    •                 复制
               var arr = [1,2,4,8,16];
                console.log('------------------');
                var _new = [];
                console.log('map实现：一个新的数组，每个元素都翻倍');
                var result = arr.forEach(function(i){ _new.push(i*i);return i *i}) //入参函数有返回值也不起作用。
                console.log(result);
                console.log(arr)
                console.log(_new);
  • [不改变原数组] —— map() 入参：Fn 必须有return 返回值。否则执行默认值返回undefined 返回： 执行过入参函数返回的新数组 不改变原数组 
    •                 复制
               var arr = [1,2,4,8,16];
                console.log('------------------');
                console.log('map实现：一个新的数组，每个元素都翻倍');
                var result = arr.map(function(i){return i *i})
                console.log(result);
