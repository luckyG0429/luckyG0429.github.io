Arrary - 常用的内置方法(2)

接着上面的那一篇，这本篇中提到的内置方法，都是在数组自身上做变更。比起上一篇的那些方法使用，我觉得更像是一个人在自己身上做实验，无论用哪种方法都会显现在自己身上。 所以本篇中的提到所有方法都会改变数组本身。在Vue 框架中也有明确指出，这些方法的使用会触发数导致视图改变.

数组的增删改

添加元素：push() | shift()

  • arr.push(param1, ... , paramN) 将一个或多个元素添加到数组的末尾，并返回该数组的新长度
  • arr.unshift(param1, ... , paramN) 将一个或多个元素添加到数组的开头，并返回该数组的新长度 
    •                 复制
               var arr = [1,4,5,10];
                console.log('------------------')
                console.log('find实现：返回大于5的数组');
                var result = arr.filter(function(i){
                         return i>=5
                });
                 console.log(result);

删除元素：shift() | pop()

  • [删除数组中的第一个元素] —— shift() 入参：空 返回：删除元素的值
  • [删除数组中的最后一个元素] —— pop() 入参：空 返回：删除元素的值 
    •                 复制
               var arr = [1,4,5,10];
                console.log('------------------')
                console.log('find实现：返回大于5的数组');
                var result = arr.filter(function(i){
                         return i>=5
                });
                 console.log(result);

splice() - 能删除，能新增，也能修改

  • [删除] —— arr.splice(下标值，要删除的元素个数) [新增] —— arr.splice(下标值，0, 添加元素值param1, param2, ...) [修改] —— arr.splice(下标值，1, 修改值) 
    •                 复制
               var arr = [1,'ni','10','hao',100, 'yeah', '!'];
                console.log('------------------');
                console.log('原数组：len='+ arr.length +';arr= [' + arr.toString() + ']');
                console.log('1.实现从10开始删除3个');
                arr.splice(2, 3);
                console.log(arr);
                console.log('2.实现倒数第二个改为hello');
                arr.splice(arr.length-2, 1, 'hello');
                console.log(arr);
                console.log('3.实现第二个元素后面加个值today');
                arr.splice(2, 0, 'today');
                console.log(arr);
  • [改变原数组] —— splice() 入参：数组下标，删除元素个数，你想添加的值 返回： 数组对象。 如果是删除，返回的被删除的值的数组。如果是添加或者在原地进行修改，返回的是空数组。 改变原数组 
    •                 复制
               var arr = ['ni','zhen','hao',10, 'kan'];
                console.log('------------------');
                console.log('splice实现：截取数组，从第2个开始，截取3个');
                var result = arr.splice(1, 3)
                 console.log(result);

遍历数组 ⤧ forEach() & map()

  • [不改变原数组] —— forEach() 入参：Fn 返回： 空 
    •                 复制
               var arr = [1,2,4,8,16];
                console.log('------------------');
                var _new = [];
                console.log('map实现：一个新的数组，每个元素都翻倍');
                var result = arr.forEach(function(i){ _new.push(i*i);return i *i}) //入参函数有返回值也不起作用。
                console.log(result);
                console.log(arr)
                console.log(_new);
  • [不改变原数组] —— map() 入参：Fn 必须有return 返回值。否则执行默认值返回undefined 返回： 执行过入参函数返回的新数组 不改变原数组 
    •                 复制
               var arr = [1,2,4,8,16];
                console.log('------------------');
                console.log('map实现：一个新的数组，每个元素都翻倍');
                var result = arr.map(function(i){return i *i})
                console.log(result);
